A clean, modern personal portfolio website for Dr. Lucas Lu, built with pure HTML, CSS, and JavaScript. Deployed on GitHub Pages.

```
lucas-portfolio/
├── index.html      # 主页面
├── style.css       # 样式文件

└── README.md       # 说明文档
```

## ✨ Features

- Responsive design — works on mobile, tablet, and desktop
- Smooth scroll animations
- Interactive navigation with active section highlighting
- Hero parallax effect
- Timeline-based experience section
- Education, publications, media coverage sections

## 🚀 Deploy to GitHub Pages

1. Push this repository to GitHub
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch → main → / (root)**
4. Your site will be live at `https://<username>.github.io/<repo>/`
